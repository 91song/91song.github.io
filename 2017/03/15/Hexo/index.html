Hexo 简单5步搭建个人博客

发表于 2017-03-15 | 分类于 Blog |

简介

快速、简洁且高效的博客框架。而且具有以下特征：

  • 超快速度
  • 支持 Markdown
  • 一键部署
  • 丰富的插件

安装

$ npm install -g hexo-cli

初始化

$ hexo init blog

切换目录

$ cd blog

生成静态文件

$ hexo generate

运行服务器

$ hexo server -s
